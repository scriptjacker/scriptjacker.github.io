Broken Access Control





    1. Check robots.txt


      Robots.txt is a text file placed on a website that provides instructions to web crawlers and search engine bots about which parts of the site should not be crawled or indexed. While robots.txt is primarily used for legitimate purposes, it can also be helpful for attackers seeking to identify potential vulnerabilities, such as those related to broken access control.

    1. Legitimate Use:

      Content Control: Website owners use robots.txt to control which parts of their website should be indexed by search engines. For example, they might want to exclude sensitive information like user profiles or admin sections from search engine results to protect user privacy or prevent sensitive data exposure.
    User-agent: *
Disallow: /user/
Disallow: /admin/
      In this case, the website owner wants to prevent search engines from indexing pages under the /user/ and /admin/ directories to protect sensitive user and administrative data.

    2. Malicious Use:

      Information Gathering Attackers can examine the robots.txt file to gain insights into the website's structure and discover hidden or restricted areas that might be vulnerable to broken access control attacks. By identifying directories or files explicitly excluded in robots.txt, attackers can target these areas for further exploitation.

      Enumeration If the robots.txt file specifies that certain directories or files are off-limits, attackers may attempt to access those resources directly, looking for misconfigurations or access control issues that could lead to unauthorized access. 
    User-agent: *
Disallow: /admin/
Disallow: /user/
Disallow: /private/
      Now, the attacker knows that the /admin/ and /user/ directories are off-limits according to the robots.txt file. The attacker then attempts to access URLs like https://example.com/admin/ and https://example.com/user/ directly. If the website has poor access controls or misconfigurations, the attacker might successfully bypass security measures and gain unauthorized access to sensitive areas.
